# PMSM Control Library

This repository contains the C implementation of the motor control algorithms developed during my MSc dissertation at the Faculty of Engineering of the University of Porto (FEUP).

## Controllers

- Field Oriented Control (FOC)
- Direct Torque Control (DTC)

## Features

- Clarke Transform
- Park Transform
- Inverse Park Transform
- PI Current Controllers
- Space Vector PWM
- Stator Flux Estimator
- Electromagnetic Torque Estimator
- Hall Sensor Processing
